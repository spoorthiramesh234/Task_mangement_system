# Task Management System

A MERN stack web application to manage tasks between Admin and Users.

## Features
- User Authentication
- Admin Task Assignment
- Task Priority (Low, Medium, High)
- Deadline Reminder
- File Upload
- Progress Tracking

## Tech Stack
Frontend: React
Backend: Node.js + Express
Database: MongoDB

## Installation

Clone repository
git clone <repo-link>

Backend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev
